# PTA適正化推進プロジェクト

本リポジトリは、全国の教育委員会に対してPTAの適正化（任意加入の徹底、公私費分離、学校施設の適正利用等）を啓発・推進するためのWebサイト管理用リポジトリです。

## 1. ディレクトリ設計の意図
公共セクターのWebサイトとして、情報の「根拠（Evidence）」を明確にし、法改正や社会状況の変化に強い構造を維持することを目的としています。

- **data/**: サイト内で引用する法令条文、判例、数値データを一元管理します。HTML内に直接記述（ハードコード）せず、ここを参照することで、情報の陳腐化を防ぎ、サイト全体の整合性を担保します。
- **docs/**: 行政機関が信頼する一次資料（文部科学省通知、裁判判決文の全文PDF等）を配置します。
- **src/**: JIS X 8341-3に準拠したセマンティックなHTMLおよびスタイルを管理します。

## 2. 情報更新・管理ルール（厳守）
情報の正確性は当委員会の信頼性の源泉です。以下のルールを遵守してください。

### 法令・判例の更新
- **出典の明記**: 法令は[e-Gov法令検索](https://laws.e-gov.go.jp/)、判例は[裁判所ウェブサイト（判例検索システム）](https://www.courts.go.jp/app/hanrei_jp/search1)を常に一次ソースとし、リンクを記述してください。
- **用語の統一**: 本サイト内では、対象を指す言葉として「生徒」を使用せず、一律で「子」または「子ども」を使用します。
- **定期点検**: 少なくとも年1回、参照している法令の改正がないか、リンク切れが発生していないかを確認してください。

### アクセシビリティの確保
- **代替テキストの必須化**: `data/slides/` 等で管理する画像データには、音声読み上げ環境を考慮し、必ず内容を正確に説明した `alt_text` を定義してください。
- **コントラストと構造**: Webデザイン変更時は、コントラスト比4.5:1以上を確保し、見出し構造（h1-h6）を論理的に構成してください。

## 3. 開発・運用フロー
1. 法令の改正や新たな判例が出た場合、まず `data/` 配下のJSONファイルを更新します。
2. その後、必要に応じて `src/` の構造を調整します。
3. 変更時は、必ず「根拠となる公的資料」のURLをコミットメッセージに含めてください。
